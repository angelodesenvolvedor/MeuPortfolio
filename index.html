Sobre Mim

Sou um desenvolvedor web sempre aprendendo, focado em criar soluções que funcionam bem e com qualidade. Gosto de tecnologia e estou sempre buscando as melhores práticas para melhorar meu trabalho. Tenho experiência com HTML, CSS, JavaScript e Next.js, criando interfaces modernas e responsivas. Uso Git para controle de versão e trabalho em equipe, e aproveito o Tailwind CSS para montar layouts bonitos e eficientes. Além disso, estou explorando TypeScript para deixar os projetos mais seguros e escaláveis.

Meus Projetos

Confira alguns dos projetos que desenvolvi:

VibePlay

Uma aplicação web interativa para reprodução de música, com recursos como controle de volume, busca por músicas, navegação por gêneros e informações sobre artistas.

FootCoder

Simulação de loja online especializada em camisetas de times de futebol. Permite navegação pelos produtos, adição ao carrinho e pagamentos como cartão, boleto bancário e PIX.

Lógica Quiz

Jogo educativo para praticar lógica proposicional, desenvolvido na disciplina de Matemática Discreta. Ajuda a aprimorar o raciocínio lógico dos usuários.

LinkTree

Minha página LinkTree com links para minhas redes sociais, portfólio e mais. Conecte-se comigo!

RunTech 🏃‍♂️🏃‍♀️

Um e-commerce focado em produtos para corrida e esportes, com layout responsivo e funcionalidades essenciais para uma boa experiência de compra.

Jogo da Velha

Jogo da Velha: O projeto oferece uma interface simples e intuitiva, permitindo que os jogadores cliquem nas células do tabuleiro para realizar suas jogadas.
